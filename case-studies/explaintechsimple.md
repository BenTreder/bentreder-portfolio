# ExplainTechSimple Case Study

## Project Type

Tech education website

## Main Focus

ExplainTechSimple is a beginner-friendly technology education website designed to explain technical topics in plain language.

## Goals

- Make technology easier to understand
- Create helpful beginner content
- Organize educational topics
- Improve search visibility
- Build a clean reading experience
- Support long-term content growth

## Key Areas

- Tech education
- Beginner guides
- Simple explanations
- Helpful content structure
- SEO-friendly pages
- Readable design

## Skills Used

- Content strategy
- Web design
- SEO
- Technical writing
- Information architecture
- User experience planning

## Results

This project demonstrates experience creating educational web content that is useful, searchable, and easy for non-technical users to understand.

# AVFoodTrucks Case Study

## Project Type

Local directory and lead generation website

## Main Focus

AVFoodTrucks is a local food truck directory and lead generation platform built for Sacramento-area food trucks, event organizers, businesses, and customers.

## Goals

- Create a local food truck directory
- Support vendor profiles
- Capture catering and event leads
- Build city and service pages
- Improve local SEO
- Create a scalable directory structure

## Key Areas

- Local directory
- Vendor profiles
- Lead forms
- City pages
- Event pages
- Admin workflows
- Local SEO

## Skills Used

- PHP
- MySQL
- HTML
- CSS
- Local SEO
- Lead generation
- Directory architecture
- Form handling

## Results

This project demonstrates experience building a practical local business platform with directory listings, lead capture, and search-focused structure.

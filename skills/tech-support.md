# Tech Support Skills

## Common Support Areas

- Slow computers
- Wi-Fi problems
- Printer issues
- Email troubleshooting
- Browser problems
- Backup setup
- Malware cleanup basics
- New computer setup
- Small office troubleshooting

## Support Approach

Good tech support should be:

- Clear
- Patient
- Practical
- Honest
- Focused on solving the actual issue
- Easy for non-technical users to understand
